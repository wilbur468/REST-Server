Player Stats Logger

CREATE NEW PLAYER

URL:

https://rest-server-playerstats.herokuapp.com/newPlayer

POST Request:

JSON DATA: {"playerName":"Cristiano Ronaldo"}

Response:

{"playerID": "T9ywf7qqB"}

ADD AGE

URL:

https://rest-server-playerstats.herokuapp.com/replace with ID of Player/age

PUT Request:

JSON DATA: {"age":37}

Response:

{"sucess": true}

ADD TROPHIES

URL:

https://rest-server-playerstats.herokuapp.com/Replace with Player ID/trophy

PUT Request:

JSON DATA: {"trophy": 32}

Response:

{"sucess": true}

GET STATS OF PLAYER

URL:

https://rest-server-playerstats.herokuapp.com/Replace with ID

PUT Request:

JSON DATA: {"age":37}

Response:

{ "Name of the Player": "Cristiano Ronaldo",
"age": 37,
"trophy": 32 }

DELETE ID OF PLAYER

DELETE REQUEST: Deletes the information of the player with associated ID

URL:

https://rest-server-playerstats.herokuapp.com/replace with id/delete

Response:

{"sucess": true}
